# 活动海报

## 适用

单场招募、系列预告、活动详情长图、活动回顾首图。

## 输入

海报类型、第一眼信息、关心理由、下一步动作、标题、价值主张、时间、地点、对象、流程、CTA、品牌素材与禁用项。

## 流程

传播目标 → 四层信息 → 视觉重心与对齐轴 → 色彩职责 → 首版 → 问题诊断 → 手机端自检。

四层信息依次是：第一眼信息、关心理由、行动信息、品牌与辅助信息。先保证第一眼信息和下一步动作，再补完整信息。

## 类型结构

- 招募：活动名与价值主张 → 时间地点与对象 → 关键权益或流程 → 报名 CTA。
- 预告：系列主题 → 首场亮点或时间线 → 首期时间与对象 → 关注或预约 CTA。
- 详情长图：标题与价值主张 → 时间地点对象 → 流程与嘉宾/内容 → 报名方式与注意事项。
- 回顾：活动结果或现场瞬间 → 关键收获 → 精选内容或数据 → 下一场 CTA。

## 视觉约束

- 标题 3 秒可读，手机端可读。
- 用规律间距建立层级；每类信息沿同一对齐轴排列。
- 主色不超过 2 个，辅助色不超过 2 个；色彩各有职责。
- 装饰不压字；避免 PPT、议程、表格、硬广感。

## 输出

类型与目标、四层文案、版式、颜色与素材职责、CTA、自检结果。
