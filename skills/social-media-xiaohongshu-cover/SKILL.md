# 小红书封面

小红书 3:4 封面专项。先把一句话定位打穿，再扩展正文和图文。

## 推荐先做封面

- 封面决定点击，反馈最快。
- 开源工具需要一句话定位，封面会逼出核心卖点。
- 封面可复用到图文第 1 页、仓库社媒预览、抖音封面。

## 输入

- 工具名。
- 一句话能力。
- 目标用户。
- 最大痛点。
- 1-3 个可证明亮点。
- 截图、logo、demo 图路径。

## 封面公式

- 痛点型：`还在手搓 X？`
- 结果型：`我把 X 变成了 Y`
- 对比型：`以前 X / 现在 Y`
- 清单型：`3 个步骤做出 X`
- 反常识型：`别先写正文，先做封面`

```text
主标题：AI 社媒 Skill 开源了
副标题：文案 / 封面 / 图文卡片 一套跑通
角标：开源工具
底部：social-media-skills
```

## 视觉方向

- 比例：3:4。
- 风格：清爽工具感 + 小红书强标题；不要纯代码仓库截图。
- 主体：三平台卡片并排，中心放工具名。
- 色彩：白底/浅灰底 + 红色强调 + 黑色大字。
- 字体：主标题粗黑，副标题中等，平台名用小标签。
