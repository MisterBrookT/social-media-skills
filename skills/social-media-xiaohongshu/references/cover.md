# 小红书封面

小红书 3:4 封面专项。归属 `social-media-xiaohongshu`，不是独立同层 skill。

## 推荐先做封面

- 封面决定点击，反馈最快。
- 开源工具需要一句话定位，封面会逼出核心卖点。
- 封面可复用到图文第 1 页、仓库社媒预览、抖音封面。

## 输入

- 工具名。
- 一句话能力。
- 目标用户。
- 最大痛点。
- 1-3 个可证明亮点。
- 截图、logo、demo 图路径。

## 封面公式

- 痛点型：`还在手搓 X？`
- 结果型：`我把 X 变成了 Y`
- 对比型：`以前 X / 现在 Y`
- 清单型：`3 个步骤做出 X`
- 反常识型：`别先写正文，先做封面`

## 开源工具默认结构

```text
主标题：AI 社媒 Skill 开源了
副标题：文案 / 封面 / 图文卡片 一套跑通
角标：开源工具
底部：social-media-skills
```

## 视觉方向

- 比例：3:4。
- 风格：清爽工具感 + 小红书强标题；不要纯代码仓库截图。
- 主体：真实产品形态、截图抽象、状态面板、卡片骨架。
- 色彩：白底/浅灰底 + 一个强调色 + 黑色大字。
- 字体：主标题粗黑，副标题中等，平台名用小标签。

## 图像提示词骨架

```text
生成一张小红书 3:4 竖版封面，主题是开源工具 [工具名]。

画面风格：[清爽/工具感/信息流强标题]。
背景：[白色/浅灰/产品主色浅底]。
主标题：[3-10 字]。
副标题：[不超过 20 字]。
角标：[开源工具/教程/方法库]。
主体：[真实截图/状态面板/卡片骨架/产品形态抽象]。

要求：主标题最大，手机信息流可读；最多三层文字；不要复杂代码截图；不要添加其他文字。
```
