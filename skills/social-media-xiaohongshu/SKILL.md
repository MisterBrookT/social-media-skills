# 小红书

小红书平台内容 skill。封面专项用 `xiaohongshu-cover`。

## 核心原则

- 先定封面承诺，再写正文。
- 像真实使用后分享，不像产品发布稿。
- 标签服务发现，不堆无关热词。
- 不写无法证明的“全网最强”“爆款必备”。

## 内容单元

- 封面：3:4，大字可读，一屏说清结果或冲突。
- 标题：20 字以内优先，关键词前置。
- 正文：300-800 字；短段、分点、真实场景。
- 标签：5-10 个；主题、人群、场景、工具、问题。
- 评论钩子：一个低门槛问题。

## 图文页纲

1. 封面：钩子 + 工具名/结果。
2. 痛点：为什么值得看。
3. 结果：做到了什么。
4. 方法：核心流程。
5. 亮点：3 个最好记的点。
6. 用法：怎么开始。
7. 收尾：适合谁 + 评论钩子。
