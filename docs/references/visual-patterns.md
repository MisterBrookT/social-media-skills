# 视觉模式

## 小红书封面

目标：手机信息流里 1 秒看懂。

常用结构：

- 大字标题 + 人物/产品主体。
- 大字标题 + 3 个标签点。
- 前后对比。
- 问题句 + 结果承诺。
- 工具/流程截图 + 强标题。

提示词应包含：

- 比例：3:4。
- 主标题：封面最大字。
- 副标题：1-2 行。
- 视觉风格：清爽、杂志、手绘、科技、职场、生活方式等。
- 主体：人物、产品、截图、场景。
- 约束：不要添加额外文字；中文无错别字；手机可读。

## 小红书图文卡片

页纲：

1. 封面：钩子。
2. 背景/痛点。
3. 核心观点。
4. 方法/步骤。
5. 案例/对比。
6. 清单/总结。
7. 行动/评论钩子。

每页一个信息点。图片承担“让人停下”的任务，正文承担解释。

## 公众号封面

输出一组：

- 21:9 主封面：完整标题或近完整标题。
- 1:1 分享图：短标题，少字，裁切安全。

不要把公众号长标题硬塞进 1:1。

## 抖音封面

目标：短、硬、可扫。

常用结构：

- 结果型：`我用 X 做成了 Y`
- 冲突型：`别再这样用 X`
- 问题型：`为什么你总是卡在 X`
- 清单型：`3 个方法解决 X`

封面标题比口播标题更短。
