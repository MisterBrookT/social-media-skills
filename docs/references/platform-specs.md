# 平台规格

## 公众号

- 核心单位：长文 + 头图 + 摘要。
- 标题：15-30 字常用；允许信息密度高，但避免空泛震惊体。
- 正文：适合 1200-3000 字；手机阅读段落短。
- 封面：常见主封面 21:9，分享/列表小图需考虑 1:1 裁切。
- 目标：建立观点、解释复杂事、沉淀方法论。

## 小红书

- 核心单位：封面 + 图文卡片 + 短正文 + 标签。
- 封面：3:4 竖图优先；大字必须手机信息流可读。
- 正文：300-800 字常用；分点、场景、真实体验更重要。
- 标签：5-10 个；覆盖主题、人群、场景、工具/品类。
- 目标：点击、收藏、评论；封面承诺必须被正文兑现。

## 抖音

- 核心单位：前 3 秒钩子 + 口播节奏 + 画面提示 + 封面标题。
- 口播：短句、强转折、可直接念；少书面连接词。
- 封面：9:16 竖图；标题更短、更直接。
- 话题：3-5 个；不用堆泛标签。
- 目标：停留、完播、互动。

## 跨平台转译

同一素材转三平台时：

- 公众号保留推理链。
- 小红书提炼经验点和可保存清单。
- 抖音抽一条冲突线，写成口播。
- 封面不共享同一标题；按平台重新起。
