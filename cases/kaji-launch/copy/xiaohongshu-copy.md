### 正文

我做了一个 macOS 菜单栏工具：`Kaji`。

它不是聊天机器人，也不是大 dashboard。

它解决一个很具体的问题：AI coding 跑久了以后，很多东西会突然打断你：

- 额度快没了
- token 压力上来了
- 系统资源飙高
- 人已经该休息了
- 长任务跑着跑着电脑睡了

Kaji 把这些状态收进菜单栏。

你不用开一个新的页面，也不用盯着复杂面板。看一眼状态，继续工作。

现在覆盖这些能力：

1. Claude Code / Codex 用量
2. 5h / 7d 重置时间
3. token 趋势和成本估算
4. CPU / 内存 / 磁盘状态
5. 专注和休息计时
6. 长任务 Keep Awake
7. Navi 熊猫状态提醒

我最想保留的是“安静”。

它应该像 Wi-Fi、电池一样待在菜单栏，不抢注意力；但关键状态要能及时被看到。

适合：macOS + AI coding 重度用户。

### 正文

`Kaji` 是一个给 AI coding 用的 macOS 菜单栏工具。

名字来自日语“舵 / かじ”：掌舵。

这个定位很准确：它不替你写代码，也不接管 workflow；它负责让你知道当前 coding session 是否还在正确航线上。

我把它做成菜单栏工具，是因为 AI coding 本来就不该再占一个大窗口。

它现在能看：

- Claude Code / Codex 用量
- 额度重置时间
- token 压力
- 估算成本
- CPU / 内存 / 磁盘
- 专注和休息节奏
- 长任务防休眠
- Navi 熊猫状态

安装方式也尽量简单：

```sh
curl -fsSL https://raw.githubusercontent.com/MisterBrookT/kaji/main/install.sh | bash
```

目前只支持 macOS 13+ 和 Apple Silicon。

项目开源，适合需要长期跑 agent、经常看用量、容易被上下文和额度打断的人。
